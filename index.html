Algorithm Rules
Priority Scheduling (Preemptive): The CPU always runs the available process with the lowest priority number (1 = most urgent). A running process is interrupted immediately when a higher-priority process arrives. Tie-break: earliest arrival time first; if still tied, alphabetical Process ID.

SRTF — Shortest Remaining Time First (Preemptive SJF): The CPU always runs the available process with the shortest remaining burst time. A running process is preempted immediately when a newly arrived process has a shorter remaining time. Tie-break: earliest arrival time first; if still tied, alphabetical Process ID.
